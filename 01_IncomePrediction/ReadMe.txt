Mini-project: Income forecasting based on demographic data
------------------------------------------------------------

Project goal: Develop a machine learning model to predict a person's income level based on demographic data
-------------

Project objectives:
-------------------
			1. Loading and analyzing the data set;
			2. Preprocessing of different types of data;
			3. Model selection and training;
			4. Model quality assessment;
			5. Visualization of results.

 
Data used: Adult Income Dataset (modified version);
----------

Problem type: Binary classification (income > $50K or <= $50K);
-------------

Technologies: ML.NET, .NET Core, C#
-------------
